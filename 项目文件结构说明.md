## 根目录结构

```
src/
├── App.js                 # 应用程序主组件，整合所有页面元素，管理全局状态和布局
├── App.css                # 应用程序主样式文件，定义全局样式和布局规则
├── index.js               # 应用程序入口文件，负责将React应用挂载到DOM
├── index.css              # 全局样式文件，定义基础样式和Tailwind CSS配置
├── reportWebVitals.js     # 性能监测工具，用于收集和报告应用性能指标
├── components/            # 组件目录，存放所有可复用的UI组件
├── assets/                # 静态资源目录，存放图片、字体等静态文件
├── lib/                   # 工具库目录，存放通用功能和辅助函数
├── config/                # 配置文件目录，存放应用配置信息
├── hooks/                 # 自定义钩子目录，存放可复用的React钩子函数
└── locales/               # 国际化文本目录，存放多语言翻译内容
```

## 主要组件功能详细说明

### App.js
应用程序的主组件，是整个应用的骨架。它负责：
- 整合所有页面区块组件
- 管理全局状态（如语言设置、页面滚动状态）
- 处理简历下载功能
- 控制启动屏幕的显示和隐藏
- 组织页面的整体布局和结构
- 提供全局事件处理（如语言切换）

### HeroSection
首屏英雄区块，是访问者看到的第一个主要内容区域：
- 展示个人姓名和职业标题
- 显示技能标签轮播
- 提供简历下载按钮
- 包含个人简介和经历描述
- 实现标签动画滚动效果
- 响应语言切换，显示对应语言内容

### Globe
3D地球组件，为页面添加视觉焦点和交互元素：
- 使用cobe库渲染3D地球模型
- 支持鼠标拖拽旋转交互
- 在地球上标记重要位置（如北京、上海、旧金山等）
- 自动缓慢旋转动画
- 点击时显示提示信息
- 适配不同屏幕尺寸

### ProjectsSection
项目展示区块，以专业方式展示个人作品：
- 以网格卡片形式展示项目
- 支持按类别筛选项目
- 实现卡片悬停动画效果
- 显示项目缩略图和基本信息
- 支持点击查看项目详情
- 响应式布局，适应不同屏幕尺寸

### ToolsSection
工具技能展示区块，展示技术专长：
- 以图标网格形式展示掌握的技术工具
- 分类展示不同领域的技能（如前端、设计工具等）
- 实现工具图标的动画效果
- 显示技能熟练度指标
- 支持语言切换，显示对应语言的技能描述

### FooterSection
页脚区块，提供联系方式和结束信息：
- 显示社交媒体链接和联系方式
- 包含版权信息和网站声明
- 提供返回顶部功能
- 实现链接悬停动画效果
- 响应式布局，适应移动设备

### FloatingDock
浮动Dock栏，提供快捷导航：
- 类似macOS的Dock，在页面底部悬浮显示
- 包含主要页面区域的快捷链接
- 实现图标悬停放大动画效果
- 根据页面滚动位置自动隐藏或显示
- 支持移动设备触摸操作

### SplashScreen
启动屏幕，提升首次加载体验：
- 应用加载时显示的动画欢迎页面
- 实现渐变动画和加载进度指示
- 完成后平滑过渡到主页面
- 提供视觉吸引力，减少用户等待感知

### Folder
文件夹组件，提供交互式的3D文件夹效果：
- 实现可点击的3D文件夹视觉效果
- 点击时展示文件夹打开动画，并弹出图片卡片
- 再次点击或滚动页面时，先收起图片卡片，再关闭文件夹
- 使用多层次结构创造深度感和立体效果
- 实现平滑的动画过渡和状态管理
- 与BounceCards组件协同工作，展示项目图片

### BounceCards
弹出式卡片动画组件，展示多张图片卡片：
- 实现卡片从文件夹弹出的动画效果
- 支持卡片悬停交互，突出显示当前卡片
- 实现卡片收起时的震动和旋转效果
- 使用GSAP动画库实现流畅的动画过渡
- 支持动画完成回调，实现与父组件的协同
- 可自定义容器尺寸、动画延迟和卡片排列

## 特色功能

1. **多语言支持**：支持中英文切换，所有界面文本和内容都能根据选择的语言动态变化，提供国际化体验。

2. **3D地球展示**：交互式3D地球展示，可以用鼠标拖拽旋转，标记了重要城市位置，增加页面的视觉吸引力和互动性。

3. **项目作品展示**：以精美卡片形式展示个人项目，支持分类筛选和动画效果，让访问者能直观了解作品集。

4. **动画效果**：丰富的动画和交互效果，包括页面过渡、滚动动画、悬停效果等，提升用户体验和视觉享受。

5. **响应式设计**：完全适配不同设备尺寸，从手机到桌面电脑都能提供良好的浏览体验，布局会根据屏幕大小自动调整。

6. **数据分析**：内置数据分析功能，追踪用户行为和交互，收集使用数据，帮助了解访客情况和优化网站体验。

## 技术栈

- **React.js**：用于构建用户界面的JavaScript库，实现组件化开发和状态管理
- **Tailwind CSS**：实用优先的CSS框架，用于快速构建自定义设计的界面
- **Framer Motion**：React动画库，用于实现流畅的页面动画和交互效果
- **Cobe**：轻量级3D地球库，用于渲染交互式地球组件
- **Canvas Confetti**：彩带效果库，用于实现点击时的庆祝动画效果
